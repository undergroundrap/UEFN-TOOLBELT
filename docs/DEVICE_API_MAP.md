# Fortnite Device API Map

This document maps the Python API for common Fortnite devices and level actors, derived from automated introspection of the UEFN level.

> [!TIP]
> Use `tb.run("api_crawl_level_classes")` to update this data for your specific level.

## Technical Details

The full schema is available in your local saved directory after running the crawler:
`%LOCALAPPDATA%\UnrealEditorFortnite\Saved\UEFN_Toolbelt\api_level_classes_schema.json`

### Example Usage

```python
import UEFN_Toolbelt as tb
# Find a lock device and lock it
for actor in tb.get_selected_actors():
    if "LockDevice" in str(actor.get_class()):
        actor.call_method("toggle_active", [False])
```
